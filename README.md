# Paperless Hospital Management System

## Description
A web-based hospital management system that eliminates paper records by digitizing appointments, prescriptions, medical history, and reports. The system also includes an AI-based emergency support chatbot for instant patient assistance.

## Domain
Healthcare Information Systems

## Features
- Role-based access (Patient, Doctor, Admin)
- Appointment booking and approval
- Digital prescriptions and medical history
- AI-based emergency support chatbot
- Secure and centralized database

## Technology Stack
- Frontend: HTML/CSS/JavaScript / React
- Backend: Node.js + Express
- Database: MongoDB / MySQL
- AI: API-based emergency chatbot

## Team Members
- Frontend Developer
- Backend Developer
- Database & AI Developer
- Documentation & Testing
